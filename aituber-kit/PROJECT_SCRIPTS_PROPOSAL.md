# 企画固定セリフ提案

## キャラクター設定
- **アイリス（A）**: 明るく元気でハイテンション、タメ口、ダジャレ・ボケを自然に挟む、天真爛漫でムードメーカー
- **フィオナ（B）**: 丁寧で柔らかい敬語、穏やかで優しい、真面目で丁寧、博識で説明する側

## 5. 肯定キーワード一覧（拡張版）

以下のキーワードを肯定と判定：

### 日本語
- はい、いい、やる、やります、やろう、やるよ、やるね
- OK、オーケー、了解、了解です、了解しました
- 承認、承認します、承認しました
- いいよ、いいね、いいです、いいですね
- やります、やる、やろう、やるよ
- 賛成、賛成です、賛成します
- お願いします、お願い、頼む
- ぜひ、ぜひとも、ぜひお願いします
- 楽しみ、楽しみです、楽しみにしてます
- やる、やる、やる、やる（繰り返し）

### 英語
- yes, yeah, yep, yup, ok, okay, sure, alright, go, let's go

### 音声認識の誤認識も考慮
- はーい、はー、やーる、やー、おーけー

## 実装時の注意事項

1. **XML形式**: 全ての固定セリフはXML形式で出力（`<A>`, `<B>`タグ）
2. **感情タグ**: 各セリフに適切な感情タグを付与
3. **文字数制限**: 掛け合いモードの文字数制限（500文字以内）を遵守
4. **ターン数**: 7ターン以上を維持
5. **キャラクター性**: アイリスはタメ口・ハイテンション、フィオナは敬語・落ち着いたトーンを維持
