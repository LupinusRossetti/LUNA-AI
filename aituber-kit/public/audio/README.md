# エフェクト用音声ファイル配置ガイド

## ディレクトリ構造

```
public/audio/
├── bgm/          # BGMファイルを配置
└── se/           # SEファイルを配置
```

## なぞなぞ企画用のSEファイル

以下のファイルを `public/audio/se/` に配置してください：

- `project-start.mp3` - 企画開始時のSE
- `correct.mp3` - 正解時のSE
- `incorrect.mp3` - 不正解時のSE

## ファイル形式

- 対応形式: MP3, WAV, OGG
- 推奨: MP3（ファイルサイズが小さいため）

## ファイル名

ファイル名は大文字小文字を区別します。以下のファイル名で配置してください：

- `project-start.mp3`（小文字）
- `correct.mp3`（小文字）
- `incorrect.mp3`（小文字）

## 注意事項

- ファイルが存在しない場合、エフェクトは再生されませんが、エラーにはなりません
- ファイルサイズが大きい場合、読み込みに時間がかかる可能性があります
- ブラウザの自動再生ポリシーにより、ユーザー操作後に再生される場合があります
